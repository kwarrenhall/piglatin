Behavior
1. For words that start with a vowel, add "ay" to the end
2. For words that start with one or more consonants, move all of the        first consecutive consonants to the end and add "ay"
3. If the first consonants include "qu", move the "u" along with the  "q".
4. For words that start with "y", treat the "y" as a consonant

Input Example
1. apple
2. cup
3. quail
4. yes


Output Example
1. appleay
2. upcay
3. ailquay
4. esyay
